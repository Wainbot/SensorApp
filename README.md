### How to install :

```bash
$ sudo npm install -g ionic cordova
$ ionic serve
```

### How to run for dev:

```bash
$ ionic serve -l
```

### How to run for emulate or prod:

```bash
$ ionic platform add [android OR ios]
$ ionic build [android OR ios]
$ ionic emulate [android OR ios]
```
